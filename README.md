# Blahajet

## About

A simple game about controlling a floating blahaj on a bumpy grasslands. Use 'WASD' to move, and watch your cute little shark eat some grass along the way. Bonus points if you can phase through the floor :D

## Play

You can compile it yourself with Godot 4.7, or go to the [itch page](https://pixelsaver.itch.io/blahajet)
